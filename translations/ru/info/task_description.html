Это вторая миссия про игру Го. Первая - Eaten Go Stones.
В этой миссии вы научитесь считать территорию в Го. Обратите внимание, что эта миссия - упрощенный вариант в сравнении с настоящими правилами игры (в тестах не будет случаев, когда камни одного цвета находятся на территории другого игрока).

Итак, что же такое территория в Го? Это все свободные пункты игрового поля, окруженные цельной и неразрывной границей, состоящей из камней одного цвета. Это может быть как форма в центре доски, формирующая границу только собственными камнями, так и форма на краю доски, которая использует этот самый край в качестве одной из границ территории. Цельная граница это такая граница, камни которой соединяются друг с другом только горизонтально и вертикально. Также граница должна быть "закрыта".
Взгляните на рисунок:

В этой ситуации есть 3 типа точек: W - территория белых, B - территория черных и N - нейтральные пункты, которые не принадлежат ни черным, ни белым (незакрытая территория, общие пункты и т.д.).
Ваша задача - посчитать, сколько территории принадлежит каждому из игроков. В данном примере ответ: {'B': 13, 'W': 12}.

Входные данные: Двумерный массив (список строк).

Выходные данные: Словарь с количеством территории каждого игрока.

Как это используется: Для разработки игрового ИИ и анализа игр.

Предусловия:
Игровое поле - 9х9, 7x7, 5x5
